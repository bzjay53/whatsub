자주 묻는 질문

Q: 자막이 표시되지 않아요.

A: 다음 사항들을 확인해보세요:

  1. 확장 프로그램이 활성화되어 있는지 확인하세요. 확장 프로그램 아이콘을 클릭하고 자막 토글이 켜져 있는지 확인합니다.
  2. 영상이 재생 중인지 확인하세요. 영상이 일시정지된 상태에서는 자막이 표시되지 않을 수 있습니다.
  3. 오디오 출력이 있는지 확인하세요. Whatsub은 오디오를 분석하여 자막을 생성합니다.
  4. 브라우저 권한 설정을 확인하세요. 확장 프로그램이 오디오 캡처 권한을 가지고 있어야 합니다.
  5. 다양한 웹사이트(YouTube, Netflix, Vimeo 등)의 모든 비디오에서 자막 기능이 작동합니다.

Q: API 키는 어디서 얻나요?

A: WhaSub은 OpenAI의 Whisper API를 사용합니다. API 키를 얻으려면:

  1. OpenAI 웹사이트에서 계정을 생성하세요.
  2. OpenAI 대시보드에서 "API Keys" 섹션으로 이동하세요.
  3. "Create new secret key"를 클릭하고 키를 생성하세요.
  4. 생성된 키를 WhaSub 설정 페이지에 붙여넣으세요.
